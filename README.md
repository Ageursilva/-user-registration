
# Sistema de Cadastro de Usuários

Aplicação front-end para cadastro e exibição de usuários utilizando MaterializeCSS e jQuery.

## Funcionalidades

- Formulário para cadastro de usuários (Nome, E-mail, Idade)
- Tabela para exibição dos dados cadastrados
- Design responsivo para desktop e mobile
- Limpeza automática do formulário após cadastro

## Tecnologias Utilizadas

- HTML5
- CSS3
- [MaterializeCSS](https://materializecss.com/) 
- [jQuery](https://jquery.com/) 
- Google Icons - Para ícones
